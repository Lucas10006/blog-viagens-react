# Blog de Viagens

Projeto desenvolvido no âmbito da unidade curricular de Interfaces Web.

## Descrição
Aplicação web desenvolvida em React que funciona como um blog de viagens, permitindo consultar posts associados a diferentes destinos.

## Tecnologias utilizadas
- React
- Vite
- JavaScript
- HTML
- CSS

## Estrutura do projeto
- `src/pages` – Páginas da aplicação
- `src/components` – Componentes reutilizáveis
- `src/App.jsx` – Componente principal da aplicação

## Funcionalidades atuais
